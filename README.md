# 🌍 Sustentabilidade no Cotidiano

Uma página web responsiva e informativa projetada para conscientizar cidadãos sobre o impacto de suas ações diárias no meio ambiente e fornecer alternativas práticas para um estilo de vida mais sustentável.

---

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido de forma nativa e limpa, utilizando:
* **HTML5:** Estruturação semântica de todo o conteúdo e dados estatísticos.
* **CSS3:** Estilização moderna, layout responsivo (Grid/Flexbox) e aplicação de imagem de fundo com máscara translúcida para otimização de leitura.

---

## ⚙️ Funcionalidades e Características

* **Design Responsivo:** A interface se adapta perfeitamente a computadores, tablets e smartphones.
* **Visual Atraente:** Fundo com imagem da natureza e paleta de cores baseada em tons ecológicos (verdes e neutros).
* **Painel de Dados:** Bloco estatístico estilizado em formato de cartões (*cards*) para destacar dados de impacto ambiental.
* **Leitura Fluida:** Contraste calibrado via CSS para evitar fadiga visual e garantir acessibilidade textual.

---

O repositório do projeto está organizado da seguinte forma:

```text
├── index.html       # Estrutura e conteúdo da página web

```

---

## 🛠️ Como Executar o Projeto

Você não precisa de servidores ou instaladores complexos para rodar este projeto localmente:

1. **Baixe ou clone** este repositório em sua máquina.
2. Certifique-se de que os arquivos `index.html` e `style.css` estejam na **mesma pasta**.
3. Certifique-se de que o arquivo HTML possui o vínculo correto para o CSS dentro da tag `<head>`:
   ```html
   <link rel="stylesheet" href="style.css">
   ```
4. Dê um **duplo clique** no arquivo `index.html` para abri-lo diretamente em qualquer navegador web moderno.

---

## 📊 Fontes de Dados Utilizadas

Os dados simulados no painel estatístico baseiam-se em panoramas gerais sobre resíduos sólidos e recursos hídricos:
* Médias de descarte e geração de resíduos sólidos urbanos.
* Indicadores gerais de desperdício em sistemas de distribuição de água tratada.
